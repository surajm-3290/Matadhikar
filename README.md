Matadhikar (meaning "Right to Vote") is a "Duolingo-style" learning application that bridges the gap between complex Election Commission of India (ECI) guidelines and the everyday voter. Built with a modern tech stack and powered by Google's Gemini AI, this platform transforms dense legal protocols into engaging, bite-sized learning loops (Read → Watch → Quiz) to boost democratic literacy.

🌟 Key Features
The 5-Phase Curriculum: Step-by-step guidance through Voter Registration (Form 6/8), Nomination Scrutiny, the Model Code of Conduct (MCC), Polling Day, and Counting Rules.

AI-Driven Dynamic Content: Powered by Google AI Studio (Gemini Flash), the app dynamically generates reading materials, video generation prompts, and interactive quizzes ensuring up-to-date and strictly neutral civic facts.

Interactive EVM & VVPAT Sandbox: A capstone simulation where users can virtually cast a vote, experience the strict 7-second VVPAT verification window, and watch the sequential counting algorithm (Postal → EVM → VVPAT Matching) unfold in real-time.

Civic Certification: Users who successfully complete the learning loop and simulation are awarded a downloadable "Matadhikar: Election Process Educator" certificate.

🛠️ Tech Stack
Frontend: React.js, HTML5, CSS3

Backend: Python (Flask/FastAPI)

AI Engine: Google AI Studio (Gemini Flash for high-speed, JSON-structured inference)

🎯 The Mission
As India continues to navigate massive democratic exercises, civic apathy and misinformation remain significant challenges. Matadhikar aims to empower first-time voters, students, and the general public with absolute clarity on their rights, the technology behind the ballot, and the strict rules that ensure free and fair elections.

# Run and deploy your AI Studio app

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
